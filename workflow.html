🤖 How to use Claude Code — no copy-pasting needed

Claude Code runs inside VS Code's terminal and edits your files directly. You describe what you want in plain English, Claude makes the change, you test it in the browser. That's it.

STEP 1 — Open terminal
In VS Code press Ctrl + ` (backtick, left of the 1 key). A terminal panel opens at the bottom.
STEP 2 — Launch Claude
Type claude and press Enter. You'll see a prompt where you can type your request.
STEP 3 — Describe what you want
Example: "add a tower that slows enemies" or "make the health bar green". Be specific about what you want to change.
STEP 4 — Claude edits the file
Claude finds the right file and makes the change automatically. No copy-pasting. You just review and test.
STEP 5 — Test in browser
Refresh localhost:5173 and try it. If it works, commit. If not, tell Claude what's wrong and it fixes it.

📦 Git — when to push and pull

git pull
⬇ Get teammates' work. Run this at the start of each session and before starting a new feature. If you skip this, you might duplicate work or get conflicts.
git add .
📌 Stage your changes. Run this after something works and you're ready to save it. The dot means "all changed files."
git commit -m "message"
💾 Save a snapshot. Write a short message describing what you did. Example: "add slow tower" or "fix enemy HP bar".
git push
⬆ Share with the team. Sends your commit to GitHub so others can pull it. Only push working code — don't push something broken.
